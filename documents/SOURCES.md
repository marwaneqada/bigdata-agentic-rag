# Sources de la base documentaire

Le projet contient des notes de synthèse originales produites à partir des supports de
cours suivants :

1. `Atelier_Apache_AIRFLOW_v1.pdf`
2. `Atelier_HDFS.pdf`
3. `Atelier_BIG_DATA._MINIO.pdf`
4. `TP 5 Traitement de flux avec Kafka Streams.pdf`
5. `TP_Spark_SQL_en (1).pdf`
6. `Atelier_BIG_DATA_spark_structured_streaming.pdf`

Les notes Markdown permettent de lancer immédiatement le projet. Pour renforcer la base
documentaire, copiez également les PDF originaux dans `documents/pdfs/`, puis relancez :

```bash
python ingest.py --reset
```
